# Reviva

**Reviva** é um projeto focado em criar fixes, ajustes e soluções para melhorar a compatibilidade de jogos modernos em placas de vídeo antigas, especialmente GPUs AMD Polaris como **RX 580, RX 570, RX 560 e RX 590**.

O objetivo do projeto é ajudar usuários que ainda utilizam hardware antigo a conseguirem executar jogos que apresentam erros de compatibilidade, travamentos ou bloqueios relacionados a drivers e recursos gráficos modernos.

---

## 🎮 Jogos compatíveis

Atualmente, o Reviva possui fixes disponíveis para:

- **Forza Horizon 6**
- **Marvel's Spider-Man 2**

A lista será atualizada conforme novos fixes forem criados e testados.

---

## 🛠️ Objetivo do projeto

O Reviva foi criado para tentar dar mais vida útil a placas de vídeo antigas, permitindo que mais pessoas consigam jogar títulos modernos sem precisar trocar de hardware imediatamente.

O projeto pode incluir:

- Correções de compatibilidade
- Ajustes para GPUs antigas
- Fixes para erros específicos
- Melhorias relacionadas a drivers
- Testes em placas AMD Polaris
- Suporte para alguns gráficos integrados AMD

---

## ✅ GPUs com foco principal

O foco principal do projeto são placas AMD da série Polaris, como:

- RX 550
- RX 560
- RX 570
- RX 580
- RX 590

Alguns fixes também podem funcionar em outras GPUs, mas isso depende de testes.
